## Install Golang

- Create Directories

  `$ mkdir $HOME/Go`
  
  `$ mkdir $HOME/Go`
  
  `$ mkdir -p $HOME/Go/src/github.com/user`

 

- Setup your paths

  `$ export GOPATH=$HOME/Go`
  
  `$ export GOROOT=/usr/local/opt/go/libexec`
  
  `$ export PATH=$PATH:$GOPATH/bin`
  
  `$ export PATH=$PATH:$GOROOT/bin`

 

- Install Go

  `$ brew install go`

 

- "go get" the basics

  `$ go get golang.org/x/tools/cmd/godoc`
